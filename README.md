# Snake Game 🐍

A Java Swing based Snake Game with:

- Authentication system
- Game session tracking
- Analytics dashboard
- Leaderboard with total points
- MySQL database integration
- Dark UI theme

## Technologies Used
- Java (Swing)
- MySQL
- JDBC
- IntelliJ IDEA

## Features
- User Login/Register
- Score Tracking
- Duration Formatting
- Leaderboard Ranking
- Analytics with date grouping
